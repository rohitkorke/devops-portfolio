# CI/CD Pipeline Project (Jenkins + Docker + AWS)

## 🚀 Overview
This project demonstrates an end-to-end CI/CD pipeline using Jenkins, Docker, and AWS EC2.

## 🛠️ Tools Used
- Jenkins
- Docker
- AWS EC2
- Maven
- GitHub

## ⚙️ Pipeline Flow
1. Code cloned from GitHub
2. Application built using Maven
3. Docker image created
4. Container deployed on EC2
